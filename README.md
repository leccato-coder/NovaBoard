# NovaBoard

NovaBoard is a modern, lightweight and highly customizable TAB, Scoreboard and Nametag plugin for Paper servers.

Designed to be fast, flexible and easy to configure, NovaBoard supports PlaceholderAPI, LuckPerms, customizable scoreboards, nametags, update checker and many advanced configuration options.

## Features

- 📋 Custom Scoreboards
- 🏷️ Custom Nametags
- 👥 TAB customization
- ⚡ Lightweight & optimized
- 🎨 RGB & Legacy colors
- 🔌 PlaceholderAPI support
- 🔐 LuckPerms support
- 🔄 Automatic Hangar update checker
- ⚙️ Easy configuration
- 🛠 Multi-version support
